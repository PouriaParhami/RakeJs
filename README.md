# RakeJs
A tool to show the number of classes and ids used

## Getting Started

Download the rake.js file and add to your html page or use RakeJs chrome extension.

## How to use
```
Rake().reportCssUsage();

```
or

```
Rake().reportCssUsageConsole();
```
